# Nova - 3D Animated Gaming Website

A modern React landing page built with GSAP animations, Tailwind CSS styling, and interactive game-inspired visuals.

## Overview

Nova is a responsive React website with animated sections, interactive overlays, and modern gaming UI. The project uses GSAP for animation, Tailwind CSS for styling, and Vite for fast development.

## Getting Started

### Prerequisites

- Node.js installed
- Git installed

### Install

```bash
git clone https://github.com/AARUSH-SING/game-website.git
cd game-website
npm install --legacy-peer-deps
```

### Run Locally

```bash
npm run dev
```

Open the local URL shown in the terminal.
